# Sohag TTS - Text to Speech Web Engine

A simple, client-side, and offline-first Text-to-Speech (TTS) web application with a premium dark-gold aesthetic.

## Features

- **Offline Voice Generation**: Works without a server connection.
- **Real-time Playback**: Listen to the generated speech instantly.
- **Customizable Voice**: Control the speed and pitch of the voice.
- **Responsive Design**: Works on desktop and mobile devices.

## How to Run

1.  Clone this repository.
2.  Open the `index.html` file in your web browser.

## Deployment on GitHub Pages

To deploy this project and make it accessible to everyone, follow these steps:

1.  **Initialize Git**: 
    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    ```

2.  **Create a GitHub Repository**: Create a new public repository on your GitHub account.

3.  **Push to GitHub**: Link your local repository to the GitHub repository and push the code.
    ```bash
    git remote add origin <YOUR_REPOSITORY_URL>
    git branch -M main
    git push -u origin main
    ```

4.  **Enable GitHub Pages**: 
    - Go to your repository's **Settings** tab.
    - In the **Pages** section, select the `main` branch as the source.
    - Click **Save**. Your site will be live at `https://<YOUR_USERNAME>.github.io/<REPOSITORY_NAME>/` in a few minutes.
